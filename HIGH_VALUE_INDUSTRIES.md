## データ収集方法

### 手動収集の手順

1. **アカウント特定**
   - 各カテゴリーで「いいね1000以上」の投稿を持つアカウントを検索
   - フォロワー数500-5000程度（個人店規模）

2. **データ抽出**
   - 最近の投稿10-20件を分析
   - 以下を記録:
     - 投稿文のコピー
     - いいね数、コメント数
     - ハッシュタグ
     - 投稿文字数
     - 絵文字の数

3. **Supabaseに手動入力**
   ```sql
   INSERT INTO engagement_metrics (
     store_id,
     category,
     category_group,
     post_content,
     hashtags,
     post_length,
     emoji_count,
     likes_count,
     comments_count,
     engagement_rate
   ) VALUES (
     NULL, -- 店舗IDはNULL（集合知専用）
     'ネイルサロン',
     '美容系',
     '今日のお客様ネイル💅✨ シンプルだけど大人可愛いデザインです🌸',
     ARRAY['#ネイル', '#ジェルネイル', '#ネイルデザイン'],
     50,
     3,
     1200,
     25,
     10.5
   );
   ```

4. **バリデーション確認**
   - 異常データ検出システムが正常に動作するか確認
   - エラーが出た場合は数値を調整
