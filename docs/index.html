Features

あなたのSNS担当がやること

01

画像を送るだけで投稿完成

LINEで写真を送ると、AIが画像を5つの視点で分析。3つの異なるコンセプトで投稿案を生成します。投稿の方向性（お知らせ/日常感/お役立ち）をボタンで選べば、さらに的確な提案に。

02

使うほど、あなたのお店らしくなる

投稿の修正や好みをAIが覚えます。使うほど、あなたのお店の書き方になります。

03

同じ業種の成功パターンを活用

同じ業種で反応が良かった投稿のパターンを自動で分析。あなたの投稿にも取り入れます。

04

反応データから自動改善 今後実装予定

いいね・保存・コメントの数を報告すると、反応が良かった投稿の特徴をAIが分析。次の投稿に自動で活かします。

05

Instagram連携でデータ同期 今後実装予定

Instagramアカウントを接続すれば、過去の投稿データ（いいね・リーチ・保存数）を自動取得。手入力なしでエンゲージメントデータを蓄積し、高精度な提案につなげます。

Output

実際に届く投稿案

写真1枚から、3つの異なる切り口で投稿案を提案します。

3案から選ぶだけ

AIが画像を分析し、3つの異なるコンセプトで投稿案を生成。「記憶に残る日常」「さりげない誘い」「店主のひとりごと」の中から、その日の気分で選べます。

本文

お店の口調に合わせた投稿文。超短文（80字）〜長文（500字）まで設定可能。

想起の一言

五感（香り・音・温度・光・手触り）を含む15〜25文字。お店を思い出させるトリガー。

来店の一文

「今」「数量」「時間」など具体情報を1つ含め、押し売りにならない来店誘導。

ハッシュタグ

同業種で高エンゲージメントのタグを自動選定。汎用タグは使わない。

撮影アドバイス

今回の写真を褒めつつ、次の投稿ネタを1つ提案。
